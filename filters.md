# Filters

| Filter | Description |
| :--- | :--- |
| **background\(color\)** | **`color`** in hexadecimal notation Set the background color of your image |
| **blur\(radius, sigma\)** | **`radius`** and **`sigma`** are numbers Blur the image with a Gaussian operator |
